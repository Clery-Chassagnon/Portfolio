LeBonCoin

Date

2019

Type

Site Web

Nombre

Groupe de 3

L'objectif de ce projet était de découvrir le framework Symfony 4. Il fallait donc en utilisant ce framework créer un pseudo Leboncoin avec une base de donnée, une connexion, une inscription, une page personnelle afin de modifier ses informations personnelles, et pouvoir consulter les informations d'une annonce ou encore poster une nouvelle annonce. Pour ce projet j'ai effectué toute la partie back end du site, le site en lui même mais aussi la liaison avec la base de donnée.
